# СептикПро — Орёл

Сайт компании по установке септиков под ключ в Орле и Орловской области.
Хостинг: Vercel (septikpro57.ru)

## Структура проекта

```
septiki-site/
├── index.html          # Главная страница (SPA)
├── uslugi.html         # Услуги
├── tseny.html          # Цены
├── o-nas.html          # О компании
├── kontakty.html       # Контакты + форма обратной связи
├── faq.html            # Часто задаваемые вопросы (аккордеон)
├── privacy.html        # Политика конфиденциальности
│
├── common.css          # Общий файл стилей для всех подстраниц
├── sw.js               # Service Worker (PWA)
├── manifest.json       # PWA манифест
├── vercel.json         # Конфигурация Vercel (редиректы, заголовки, CSP)
├── sitemap.xml         # Карта сайта для SEO
├── robots.txt          # Инструкции для поисковых роботов
│
├── img/gallery/        # Фото для галереи (добавить при наличии)
└── og-image.svg        # Open Graph изображение
```

## Страницы

| URL | Назначение |
|-----|-----------|
| / | Главная: Hero, преимущества, галерея, отзывы, калькулятор, цены, FAQ, контакты |
| /uslugi | Услуги компании |
| /tseny | Цены и гарантии |
| /o-nas | О компании |
| /kontakty | Контакты + форма обратной связи |
| /faq | Часто задаваемые вопросы (аккордеон, 4 категории) |
| /privacy | Политика конфиденциальности (noindex) |

## Ключевые функции

- **Калькулятор** — автоматический расчёт стоимости в зависимости от количества человек, типа грунта и расстояния
- **Галерея** — lazy-loading, фильтр по категориям, lightbox для просмотра
- **Форма обратной связи** — отправка заявок через Google Apps Script
- **PWA** — Service Worker для офлайн-доступа
- **SEO** — JSON-LD разметка, Open Graph, Яндекс.Метрика, Google Tag Manager
- **Анимации** — IntersectionObserver для появления элементов (reveal)

## Установка и запуск

```bash
# Просто открыть index.html в браузере
# или развернуть на Vercel
npx vercel --prod
```

## Навигация

- На всех подстраницах используется единый header с навигацией и мобильное меню (hamburger)
- В footer — полная карта сайта со ссылками на все страницы
- Для редиректов без .html настроены в vercel.json
