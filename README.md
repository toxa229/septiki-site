# СептикПро — Орёл

Продукты для компании по продаже, установке и обслуживанию септиков под ключ в городе Орёл.

## Структура проекта

```
septik-orel/
├── README.md                     # Этот файл
├── .env.example                  # Пример конфигурации Telegram бота
│
├── telegram-bot/                 # Telegram бот (Python aiogram 3.x)
│   ├── main.py                   # Точка входа, запуск бота
│   ├── config.py                 # Настройки, константы, прайсы
│   ├── requirements.txt          # Зависимости Python
│   ├── handlers/
│   │   ├── client.py             # Хендлеры клиента (калькулятор, заказы, объекты, чат)
│   │   ├── master.py             # Хендлеры мастера (чек-лист, фото, геолокация)
│   │   └── admin.py              # Админ-панель (назначение заявок, мастера, статистика)
│   ├── keyboards/
│   │   ├── client_kb.py          # Клавиатуры клиента
│   │   └── master_kb.py          # Клавиатуры мастера
│   ├── database/
│   │   ├── models.py             # SQLite — создание таблиц
│   │   └── queries.py            # Все SQL-запросы
│   └── utils/
│       ├── calculator.py         # Расчёт стоимости септика
│       └── notifications.py      # Уведомления клиентов и мастеров
│
└── website/                      # Tilda-сайт (Zero Block HTML)
    ├── tilda-hero.html           # Hero-секция + мини-калькулятор
    ├── tilda-advantages.html     # Блок преимуществ (6 карточек)
    └── content-template.txt      # Структура контента для всех страниц
```

---

## 1. Сайт (Tilda)

### Быстрый запуск

1. Зарегистрироваться на **tilda.cc** (тариф Pro — ~6000₽/мес)
2. Создать новый проект → Zero Block
3. Вставить HTML-код из файлов:
   - `website/tilda-hero.html` — главный экран
   - `website/tilda-advantages.html` — блок преимуществ
4. Наполнить контентом по шаблону `website/content-template.txt`
5. Подключить домен (septik-orel.ru)
6. Настроить Яндекс.Метрику и Яндекс.Бизнес

### Ключевые страницы
| URL | Назначение |
|-----|-----------|
| / | Главная с калькулятором |
| /uslugi | Услуги |
| /katalog | Каталог септиков |
| /kalkulyator | Полный калькулятор |
| /portfolio | Примеры работ |
| /tseny | Цены |
| /blog | SEO-статьи |

## 2. Telegram бот

### Быстрый запуск

```bash
cd telegram-bot

# Установка зависимостей
pip install -r requirements.txt

# Настройка
cp ../.env.example ../.env
# Отредактировать .env — указать BOT_TOKEN

# Запуск
python main.py
```

### Возможности

#### Клиент:
- 🧮 Калькулятор стоимости септика (жильцы × грунт)
- 📞 Заказ бесплатного замера
- 📋 Отслеживание статуса заявок
- 🏠 Мои объекты с прогресс-баром этапов
- 📅 График обслуживания (ТО)
- 💬 Чат с поддержкой

#### Мастер:
- 📋 Чек-лист работ с отметкой выполнения
- 📸 Фотоотчёт до/после
- 📍 Геолокация объекта
- 🔄 Смена статуса заявки
- 💬 Чат с клиентом

#### Админ:
- 🆕 Назначение мастеров на заявки
- 👷 Управление мастерами
- ⚠️ Мониторинг просроченных ТО
- 📊 Статистика
