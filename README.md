# СептикПро — Автономная канализация под ключ

Одностраничный сайт-визитка для компании по монтажу септиков.

## Возможности
- 📱 Адаптивный дизайн (мобильные + десктоп)
- 💬 Встроенный чат-бот с базой знаний
- 📋 Форма заявки (Email + Telegram)
- ⚡ Быстрая загрузка

## Разработка
Просто отредактируйте `index.html` и закоммитьте — сайт обновится автоматически.
