Политика конфиденциальности

Настоящая Политика конфиденциальности персональных данных (далее — Политика) действует в отношении всей информации, которую компания СептикПро (ИП, далее — Оператор) может получить о пользователе во время использования сайта septikpro57.ru.

1. Общие положения

1.1. Используя сайт septikpro57.ru, пользователь соглашается с условиями настоящей Политики.

1.2. Если пользователь не согласен с условиями Политики, он должен прекратить использование сайта.

2. Какие данные собираются

2.1. Персональные данные, предоставляемые пользователем:

  • Имя и номер телефона (при заполнении формы обратной связи)
  • Адрес электронной почты (при отправке заявки)
  • Сообщение, которое пользователь оставляет в форме

2.2. Автоматически собираемые данные:

  • IP-адрес
  • Тип браузера и устройства
  • Дата и время посещения
  • Страницы, которые просматривает пользователь
  • Данные Яндекс.Метрики и Google Analytics (в обезличенном виде)

3. Цели сбора данных

3.1. Персональные данные собираются для следующих целей:

  • Обработка заявок и обратный звонок пользователю
  • Консультирование по услугам компании
  • Улучшение качества сайта и услуг
  • Аналитика посещаемости (Яндекс.Метрика, Google Analytics)

4. Обработка и хранение данных

4.1. Обработка персональных данных осуществляется на законной основе.

4.2. Срок хранения персональных данных — не дольше, чем этого требуют цели обработки.

4.3. Персональные данные не передаются третьим лицам, кроме случаев, предусмотренных законодательством РФ.

5. Защита данных

5.1. Оператор принимает необходимые организационные и технические меры для защиты персональных данных от неправомерного доступа, уничтожения, изменения, блокирования, копирования и распространения.

5.2. Передача данных осуществляется по протоколу HTTPS с шифрованием.

6. Файлы cookie

6.1. Сайт использует файлы cookie для работы систем аналитики (Яндекс.Метрика, Google Analytics).

6.2. Пользователь может отключить cookie в настройках браузера.

7. Права пользователя

7.1. Пользователь имеет право:

  • Получить информацию о том, какие данные о нём хранятся
  • Требовать удаления или изменения своих персональных данных
  • Отозвать согласие на обработку персональных данных

7.2. Для реализации своих прав пользователь может направить запрос на email: info@septikpro57.ru

8. Изменение Политики

8.1. Оператор имеет право вносить изменения в настоящую Политику. Новая редакция вступает в силу с момента её размещения на сайте.

8.2. Актуальная версия Политики всегда доступна по адресу: septikpro57.ru/privacy.html
